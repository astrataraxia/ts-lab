# Studio/01

작은 인터랙티브 실험을 모아두는 TypeScript 프로젝트입니다.

## 시작하기

```powershell
npm install
npm run dev
```

## 주요 기능

- Home: 프로젝트 진입 허브
- Sound Canvas: 오디오 데이터를 이용한 Canvas 시각화
- Game Lab: 추후 추가 예정

## 프로젝트 구조

디렉터리 구조와 확장 규칙은 [아키텍처 문서](docs/ARCHITECTURE.md)를 참고하세요.

## 검증

```powershell
npm run format
npm run check
npm run build
```
